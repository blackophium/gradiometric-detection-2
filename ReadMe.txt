Analiza mechanicznej aktywności serca w oparciu o pomiar gradiometryczny czujnikiem indukcyjnym, przy użyciu urządzenia NWT6000.

1. Wykonaj pomiar 'Pomiar.py' ustalając liczbę kroków
2. Dane zostaną zapisane do pliku CSV
3. Wprowadz dane do oprogramowania Matlab i użyj funkcji Analiza lub Analiza_r (FFT, filtracja próbek, odwrotna FFT) 
4. Przedstawiony wykres to zapis w dziedzinie czasu mechanicznej aktywności serca w [s]



Analysis of the mechanical activity of the heart based on the gradiometric measurement with an inductive sensor, using the NWT6000 device.

1. Carry out the measurement 'Measure.py' determining the number of steps
2. The data will be saved to a CSV file
3. Enter the data into Matlab and use the Analyze or Analysis_r functions (FFT, sample filtration, inverse FFT)
4. The presented graph is a record in the time domain of the mechanical activity of the heart in [s]
